Goal:
Build this project feature by feature without copying solutions.

1. Add Student
2. Display Students
3. Search by ID
4. Update Marks
5. Delete Student
6. Find Top Student

Recommended order:
Day 1: Add + Display
Day 2: Search
Day 3: Update
Day 4: Delete
Day 5: Top Student + Cleanup
